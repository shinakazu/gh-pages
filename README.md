# TODO App

シンプルな TODO アプリケーションです。HTML / CSS / JavaScript のみで構成されており、サーバー不要でブラウザ上で動作します。

## 機能

- タスクの追加・削除
- タスクの完了/未完了の切り替え
- フィルタリング（All / Active / Completed）
- 完了済みタスクの一括削除
- LocalStorage によるデータ永続化

## 使い方

`index.html` をブラウザで開くだけで利用できます。

```bash
open index.html
```

## データ保存について

タスクのデータはブラウザの LocalStorage に JSON 形式で保存されます。サーバーやクラウドには送信されず、すべてローカルのブラウザ内で完結します。

- 保存先はブラウザ内部のファイルシステム（OS 上のブラウザプロファイルディレクトリ）
- オリジン単位（プロトコル + ホスト + ポート）で隔離される
- ブラウザのデータを削除するとタスクも消える
- 容量の上限は通常 5〜10MB 程度
